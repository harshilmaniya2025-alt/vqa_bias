## To run this notebook
1. Download the VQA v2 dataset subset.
2. Place it in a `data/` folder next to `vqa_bias.ipynb`, with this structure:
   - data/question/subset_q_train.json, subset_q_val.json
   - data/annotation/subset_anno_train.json, subset_anno_val.json
   - data/train_subset/ (COCO train2014 images)
   - data/val_subset/ (COCO val2014 images)
3. Open vqa_bias.ipynb in Jupyter and run all cells.
